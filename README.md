# 🛍️ AR Shopping Assistant

An interactive mobile application developed using Unity that allows users to scan products and view their details such as name, price, and description.

## 🚀 Features
- User Login & Signup System
- Product Scanning using Camera/Gallery
- Displays Product Name, Price, Description
- Simple and User-Friendly UI
- Electronics Product Dataset (10 items)

## 🧠 How It Works
- User clicks "Scan"
- Camera/Gallery opens
- Image is selected
- App identifies product using rule-based matching
- Displays corresponding product details

## 🛠️ Technologies Used
- Unity (C#)
- TextMeshPro (UI)
- Native Gallery Plugin
- Git & GitHub

## 📱 Future Enhancements
- AI-based real image recognition

- Online product database integration
